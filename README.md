## Azenso
### Modular Inventory and Finance Assistant for Small Business

-----
#### Overview:
Azenso is a financial assistant application designed for but not limited to small business owners. This application allow the users to monitor the flow of money by saving and managing inventories, expenses, and savings.

-----
#### Objectives:
This project aims to:

- Reduce manual writing and tracking errors
- Help small sellers monitor inventory and operating expenses
- Provide clear Profit & Loss (PnL) reports
- Support savings planning for business growth
- Offer industry-specific inventory modules in one unified system

-----
#### Key Features:

This project functionalities includes:

- Analytical Dashboard that monitors the trend and profit over time
- PnL system that monitors profit gain and losses from the sold and damaged products
- inventory system that allow users to select which category of business they are managing
- expenses system that is also divided into category depending on which type of expenses the users will be monitoring
- saving system with different functionality depending on which system is suitable to your case

-----
#### Technology used:

- Android Studio (Kotlin/xml layout)
- Sqlite database
- Material UI Components
